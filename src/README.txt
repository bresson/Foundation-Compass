ie - IE Shiv Styles
media_queries - mobile, screen and print
mixins - compass/sass functions for doing px/rem font, padding and etc
modules - module/widget specific styles
partials - all gobal basic styles such as tables, lists typography and etc
themes -  your color themes for creating new things
			switching themes is a matter of changing the body class to the theme name

partials/settings.scss - Contains all the settings for your site, number of column grid layout, max/min width of main container and etc

styles.scss - all of the files listed in here are compiled into one css file called styles in the cache dir css

themes.scss - all of your themes get compiled into one css file called themes.css in the css cache dir.
